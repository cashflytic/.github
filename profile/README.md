# Cashflytic

## Smarter Expense Management

Cashflytic is a fintech platform for **automated expense tracking, receipt digitisation, and spend analytics** — built to give teams real-time visibility into their spending.

### What We Build

- **Receipt OCR** — capture and digitise receipts automatically using AI
- **Uber Integration** — import business trips directly into your expense reports
- **Spend Analytics** — track, categorise, and report on expenses in real time
- **Financial Domain Logic** — a clean, domain-driven core that keeps business rules honest

### Technology

Java 21 · Spring Boot 3 · PostgreSQL · TypeScript · Mistral AI · Docker
